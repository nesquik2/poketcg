# poketcg

A PokémonTCG-inspired game except it's already rigged because it only includes Pokémon I like. You can also draw and create your own cards.

>  Card images are currently placeholders! All UI, assets, and artwork are drawn by me, and the goal is for every card to eventually use original custom artwork...

## Running the App

Download the latest release from the Releases page and launch the app

## Development

Install dependencies:

```bash
npm install
```

Start the React development server:

```bash
npm start
```

In a second terminal, launch the Electron app:

```bash
npm run electron
```

## Building

Create a production build:

```bash
npm run dist
```
